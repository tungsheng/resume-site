API NOTES (verified from installed satteri@0.9.1 dist types — kept for ref)
----
- Plugin helpers: defineMdastPlugin / defineHastPlugin from "satteri".
  Register via satteri({ mdastPlugins:[...], hastPlugins:[...] }).
- Admonition (#16): defineMdastPlugin({ name, blockquote(node, ctx) }).
- Blog images (#17): defineHastPlugin({ name,
    element: { filter:['img','p'], visit(node, ctx) } }).
